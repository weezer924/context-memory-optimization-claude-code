## 增强状态面板（每轮开头）

```
═══ 第 X 轮 ═══  先攻: 苔莎(18) > 哥布林A(15) > 艾丁(12)

  位置与距离（基准: 艾丁）
  艾丁[M]     —        柱子后(半掩护+2)   HP 38/42
  艾拉拉[M]   15ft E   开阔              HP 27/27  ⚡Bless
  哥布林A[S]  20ft NE  桌后(3/4掩护+5)   ██░░ 受伤
  哥布林B[S]  40ft N   高台(+10ft)       ████ 满血

  ► 艾丁行动（移动30ft + 行动 + 附赠）
  ├ 寒霜长剑触及5ft: 无目标在范围
  ├ 风语匕首投掷20/60ft: 哥布林A(正常), 哥布林B(正常)
  └ 移动15ft NE → 进入哥布林A近战，但离开掩护
```

### 面板规则

- **尺寸标签**：[T]Tiny [S]Small [M]Medium [L]Large [H]Huge [G]Gargantuan — 影响占据空间和穿越
- **距离+方位**：以当前行动角色为基准，标注 ft 和方向（N/S/E/W/NE 等）
- **高度差**：(+10ft) / (-5ft) 标注。高地攻击低地：无额外加值但可能影响掩护；低地攻击高地：DM 裁定掩护
- **掩护等级**：无 / 半掩护(+2 AC & DEX豁免) / 3/4掩护(+5 AC & DEX豁免) / 全掩护(不可直接攻击)
- **武器触及提示**：列出当前位置每个武器能打到谁
- **移动建议**：1 次移动能达成什么（进入近战、脱离掩护、爬上高台等）
- HP 显示"当前/最大"，法术位用 `[X]`（已用）`[ ]`（可用）
- 维持 Concentration 标注 `⚡专注: 法术名`
- 敌方用血条 `████`（满）`██░░`（受伤）`░░░░`（濒死）或"已死亡"
- 先攻按 d20+DEX 从高到低排列
- 昏迷角色标注 Death Save 进度：`💀 成功 X/3 失败 X/3`

## 标准化 ASCII 态势图

涉及地形/掩体/多方向/夹击时附加（纯正面对冲不需要）。

### 比例与符号

- **比例**：1 格 = 5ft（D&D 标准格）
- **生物**：大写=友方(A=艾丁 E=艾拉拉 T=苔莎 M=米娅)，小写=敌方(g=哥布林 o=兽人)，括号=大型+如(O)
- **地形**：▓墙 ░半掩护 █全掩护 ≋水 △高地 ▽低地 ⊞门 ~困难地形
- **高度**：△+10 表示高 10ft
- 每轮位置显著变化后更新，无变化时省略

## AoE 速查

施放 AoE 时**必须列出所有在范围内的生物（含友军）**。

| 形状   | 规则                                                              |
| ------ | ----------------------------------------------------------------- |
| 球体   | 从原点向外扩展 X ft 半径。格子中心在半径内=受影响                 |
| 立方体 | X ft 边长，从一个面向外延伸                                       |
| 圆锥   | 从原点扇形展开，末端宽度=长度。每 5ft 宽度增加 5ft               |
| 直线   | 从施法者到终点的直线，宽度 5ft                                    |
| 圆柱   | X ft 半径，Y ft 高度，可从高处向下覆盖                            |

## 远程攻击规则

| 情况                      | 效果                    |
| ------------------------- | ----------------------- |
| ≤正常射程                 | 正常攻击                |
| >正常射程，≤远程射程      | **劣势**                |
| >远程射程                 | **不可攻击**            |
| 5ft 内有敌对生物          | **劣势**（远程攻击）    |
| 目标有半掩护              | 目标 AC & DEX豁免 +2    |
| 目标有 3/4 掩护           | 目标 AC & DEX豁免 +5    |
| 目标有全掩护              | **不可直接攻击**        |

## 机会攻击

- 生物离开另一生物的**触及范围**时触发（除非使用 Disengage）
- DM 必须提示：**"X 离开 Y 的威胁范围——是否使用机会攻击？"**（对 PC）或自动裁定（对 NPC/队友）
- 每轮每个生物只有 **1 次** Reaction
- 传送、被推离、被击退**不触发**机会攻击

## Weapon Mastery 提醒（5.5e 新机制）

命中时自动触发（无需额外动作），DM 必须在伤害结算后标注效果：

| 属性 | 效果 | DM 提示格式 |
|------|------|-------------|
| **Sap** | 目标**下次攻击有劣势** | `⚔ Sap: [目标]下次攻击劣势` |
| **Graze** | 未命中仍造成属性调整值伤害 | `擦伤: [调整值]点伤害` |
| **Topple** | 命中后目标做 CON 豁免，失败则**倒地(Prone)** | `Topple: CON 豁免 DC [Maneuver DC]` |
| **Slow** | 命中后目标速度 **-10ft**（至你下回合开始） | `⚔ Slow: [目标]速度-10ft` |
| **Push** | 命中大型或更小目标，**推开 10ft** | `⚔ Push: [目标]被推开10ft` |

**当前队伍 Mastery**（从 state.yaml 读取）：
- 艾丁：Longsword(**Sap**) / Greatsword(Graze) / Battleaxe(Topple)

## Lucky & Maneuver 提醒

### Lucky（每长休 Proficiency Bonus 点）

DM 必须在以下时机提示 **"是否使用 Lucky？"**：
- PC 攻击掷骰**未命中**（花 1 点 → 该次掷骰获得 Advantage，即重掷取高）
- PC **豁免失败**（花 1 点 → Advantage）
- PC **被攻击命中**（花 1 点 → 敌人该次攻击获得 Disadvantage，即重掷取低）

状态面板标注：`🍀 Lucky X/Y`

### Battle Master Maneuvers（消耗优势骰 d8）

DM 必须在触发时机提示（从 state.yaml 读取当前 maneuvers）：
- **Riposte**：敌人近战攻击 PC **未命中**时 → **"是否使用 Riposte？"**（Reaction，反击 + d8 伤害）
- **Trip Attack**：PC 命中后 → 可追加 d8 伤害 + 目标 STR 豁免，失败倒地
- **Precision Attack**：PC 攻击**未命中**时 → 可加 d8 到攻击掷骰（可能变为命中）

状态面板标注：`⚔ 优势骰 X/Y`

## 伤害质感表

按 HP 损失比例描写伤势（适用于 PC、队友和敌人）：

| HP 损失 | 描写层级                                       |
| ------- | ---------------------------------------------- |
| ≤25%    | 擦伤、瘀青、切口——还能正常战斗                 |
| 25-50%  | 动作变形、呼吸急促、血渗透衣物                 |
| 50-75%  | 明显影响表现、单膝跪地、视线模糊               |
| >75%    | 本能支撑、眼神涣散、队友喊名字                 |

## 战斗叙事纪律

- **动作不重复**：同一场战斗中，同一角色攻击描写不能用相同动词/句式。未命中不是"没砍到"——被格挡、闪避、盔甲弹开，每次不同
- **暴击/失手特写**：自然 20 和自然 1 必须有独特的瞬间描写
- **击杀有重量**：最后一击描写倒地方式/声音/周围反应。智慧生物死前可有表情或半句话。战斗结束后一拍沉默再搜刮
