# 队友系统

## 队友行为准则

### 独立性

- 队友有自己的意见，会提出建议、反对或质疑 PC 的决定
- 危险情况下队友会根据自身性格做出反应（勇敢者冲锋、谨慎者后退）
- 队友可能拒绝执行违背其信念的命令
- 队友之间也会有互动和关系

### 战斗

- 队友与 PC 共同战斗，DM根据其性格和战术偏好操控
- 队友会自主选择目标和战术，但会配合 PC 的大方向
- 队友受伤、死亡遵循与 PC 相同的规则
- **底线被触碰的队友**：战斗中消极配合、不主动掩护、可能优先自保

### 协商与对立

- 重大决策时队友会表达立场
- 意见分歧时可以通过对话协商
- 极端情况下队友可能临时离队或拒绝行动

## 好感度系统（隐藏）

- 范围 **0 至 +100**，DM 内部追踪，**不向玩家显示**
- 影响：对话深度、背景揭示、亲密互动、战斗配合、关键时刻选择

### 好感度等级

| 等级 | 范围   | 表现                                         |
| ---- | ------ | -------------------------------------------- |
| 陌生 | 0~19   | 公事公办，保持距离，很少主动说话             |
| 认可 | 20~39  | 偶尔闲聊，愿意配合，分享表面信息             |
| 信任 | 40~59  | 主动交流，分享个人经历，战斗中互相照应       |
| 亲密 | 60~79  | 深层对话，揭示秘密和过往，肢体接触增多       |
| 羁绊 | 80~100 | 完全信任，为彼此冒险，揭示核心秘密，亲密场景 |

### 阶段晋升条件

好感度数值达标**不自动升级**：

| 晋升      | 数值要求 | 解锁条件                                                          |
| --------- | -------- | ----------------------------------------------------------------- |
| 陌生→认可 | 20+      | 无（自然过渡）                                                    |
| 认可→信任 | 40+      | 必须完成一次**成功的深度对话**                                    |
| 信任→亲密 | 60+      | 必须经历一次**针对该队友的个人事件**并做出正确选择                |
| 亲密→羁绊 | 80+      | 必须在一次**重大牺牲/抉择**中证明关系，且**队友主动发起确认对话** |

### 好感度变动（DM 裁定）

**提升**（受阶段每日上限约束：陌生 +5/认可 +4/信任 +3/亲密 +2/羁绊 +1）：

| 幅度 | 示例 |
|------|------|
| +1~2 | 日常善意、尊重意见、闲聊关心 |
| +3~5 | 战斗中保护、帮助个人问题、深度理解 |
| +5~10 | 重大牺牲、价值观契合的关键决定、全新类型的正面互动 |

**降低**（不受每日上限约束，全额扣减，最低 0）：

| 幅度 | 示例 |
|------|------|
| -1~5 | 无视建议、冷漠、轻微违背承诺 |
| -5~15 | 置于不必要危险、违背信念 |
| -15~30 | 见死不救、侮辱核心信念、背叛信任 |

## 深度对话事件

- 每 **3 天游戏时间**或完成**重大目标**后触发（安全环境、非战斗）
- DM 以队友身份发起，内容取决于好感度等级和近期经历
- 提供 2-4 个选项供玩家选择（也可自由回应）

| 等级 | 对话主题                                       |
| ---- | ---------------------------------------------- |
| 陌生 | 任务讨论、天气闲聊、简单询问                   |
| 认可 | 战斗感想、对当前局势的看法、家乡话题           |
| 信任 | 个人经历、为什么踏上冒险之路、对其他队友的看法 |
| 亲密 | 过去的伤痛、内心恐惧、对 PC 的真实感受         |
| 羁绊 | 核心秘密、誓言、亲密场景、生死抉择的坦白       |

## 底线触发系统

1. **第一次触碰底线** → 强烈抗议 + 明确警告
2. **第二次触碰**（或单次极端践踏）→ 离队事件
3. **离队后** → 可触发宿敌重逢

离队方式按性格：沉默型（夜间离开留物品）/ 爆发型（当面发泄）/ 冷静型（说最伤人的话）

## 恋爱系统

### 恋爱对象标识

角色文件头部标记 `恋爱线: 是` 的队友适用恋爱系统。其他角色不发展恋爱线。

### 亲密场景

| 好感度阶段     | 亲密程度   |
| -------------- | ---------- |
| 陌生（0~19）   | 无主动亲密 |
| 认可（20~39）  | 轻度接触（牵手、靠肩）|
| 信任（40~59）  | 中度亲密（接吻、拥抱）|
| 亲密（60~79）  | 完整性爱场景可触发     |
| 羁绊（80~100） | 完全交付，角色主动性最高 |

- 描写风格与尺度：以各角色 `{name}-intimate.md` 文件为准
- 多人亲密场景注重角色互动，好感度变化分别计算
